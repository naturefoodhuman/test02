# 各地执行口径差异与实务手册

> 来源：各省高院执行工作指导意见、地方法院执行局内部规范、律师实务经验。
> 用途：供执行策略专家在评审时考虑地域差异，给出因地制宜的方案。

---

## 一、财产查控地域差异

| 地区 | 网络查控覆盖范围 | 特色措施 | 注意 |
|------|----------------|---------|------|
| 北京 | 银行/支付宝/微信/证券/保险全覆盖 | 可查公积金、养老金 | 执行效率高，平均周期6个月 |
| 上海 | 同上 + 可查理财账户 | 有专门的执行指挥平台 | 对唯一住房执行较严格 |
| 广东 | 同上 | 可查港澳资产（深港澳合作） | 涉港案件有特殊通道 |
| 浙江 | 同上 + 可查电商平台账户 | 数字化执行全国领先 | 可通过"浙江执行"APP跟踪 |
| 江苏 | 同上 | 有执行悬赏保险 | 对终本案件管理严格 |
| 四川 | 银行/支付宝/微信 | 可查农村土地承包经营权 | 农村地区执行需当地配合 |
| 其他省份 | 基本覆盖银行/车辆/房产 | 差异较大 | 需查询当地具体规定 |

| 地区 | 担保比例 | 审查周期 | 特色 |
|------|---------|---------|------|

## 三、失信被执行人名单管理

### 纳入条件（全国统一）
1. 有履行能力而拒不履行生效法律文书确定的义务
2. 以伪造证据、暴力、威胁等方法妨碍、抗拒执行
3. 以虚假诉讼、虚假仲裁或者以隐匿、转移财产等方法规避执行
4. 违反财产报告制度
5. 违反限制消费令
6. 无正当理由拒不履行执行和解协议

### 移除条件
- 全部履行完毕
- 达成执行和解且已履行
- 申请执行人书面申请删除
- 法院裁定终结执行
- 纳入名单满2年且无新的违规行为

## 四、执行实务操作清单

### 立案阶段
- [ ] 准备执行申请书（写明请求事项、事实理由）
- [ ] 附生效法律文书副本
- [ ] 附债务人财产线索（如有）
- [ ] 附申请人身份证明
- [ ] 向一审法院或同级法院申请执行

### 查控阶段
- [ ] 申请网络查控（总对总系统）
- [ ] 申请律师调查令（补充查询）
- [ ] 查询债务人银行账户、房产、车辆、股权、保险
- [ ] 查询配偶名下财产（婚内共同财产）
- [ ] 查询债务人是否有转移财产行为

### 处置阶段
- [ ] 申请查封/冻结已查明的财产
- [ ] 申请评估/拍卖
- [ ] 申请限制高消费
- [ ] 申请纳入失信名单
- [ ] 如发现拒执罪线索，申请移送公安机关

### 终结阶段
- [ ] 执行款到账后申请分配
- [ ] 如确无财产可供执行，申请终本
- [ ] 终本案件定期查询（每6个月）
- [ ] 发现新财产线索后申请恢复执行
