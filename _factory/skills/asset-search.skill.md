## knowledge

### 可调查财产类型清单

#### 1. 银行存款
- **查询方式**：法院"总对总"网络查控系统 / 律师调查令
- **覆盖范围**：全国所有商业银行、农村信用社、邮储银行
- **执行方式**：直接冻结 → 划扣
- **注意**：需知开户行信息（可通过交易流水反查）

#### 2. 房产
- **查询方式**：不动产登记中心（律师持调查令）/ 法院网络查控
- **执行方式**：查封 → 评估 → 拍卖
- **唯一住房**：可以执行，但需保障被执行人基本居住需求（留出租金）
- **注意**：查档需知身份证号和姓名

#### 3. 车辆
- **查询方式**：车管所 / 法院网络查控
- **执行方式**：查封 → 扣押 → 拍卖
- **注意**：车辆可能已转卖但未过户

#### 4. 股权/投资
- **查询方式**：国家企业信用信息公示系统 / 企查查 / 天眼查
- **执行方式**：冻结股权 → 强制转让
- **注意**：需确认股权是否已质押

#### 5. 保险
- **查询方式**：法院向各保险公司发协助执行通知书
- **可执行部分**：保单现金价值（退保后可执行）
- **注意**：人身保险的保险金部分不可执行

#### 6. 公积金
- **查询方式**：公积金管理中心（律师持调查令）
- **执行方式**：符合条件时可提取执行
- **注意**：各地政策不同，需查询当地规定

#### 7. 支付宝/微信
- **查询方式**：法院向支付宝/腾讯发协助执行通知书
- **执行方式**：冻结余额 → 划扣
- **注意**：余额宝/理财通等理财产品同样可执行

#### 8. 证券账户
- **查询方式**：中国结算公司 / 法院网络查控
- **执行方式**：冻结 → 强制平仓或转让
- **注意**：需知证券公司名称

### 调查顺序建议
1. 先查银行存款（最快、最直接）
2. 同时查房产/车辆（价值大、易执行）
3. 再查支付宝/微信（年轻人常用）
4. 最后查股权/保险/公积金（补充性调查）
