# ADR-002：策略推理 GLM 优先 + 法条防幻觉
- 状态：已接受
- 日期：2026-06-12
- 背景：法律推理对准确性要求极高，LLM 易"编造法条/程序"（法律幻觉）。
- 决策：策略推理默认 GLM(cloud/glm-primary)、本地 qwen 兜底；法条强制来自国家法规库，生成后做忠实度校验，附免责声明+建议律师复核。
- 备选：纯本地模型（弃因：复杂法律推理质量不稳）；LLM 直出法条（弃因：幻觉风险）。
- 后果：质量更稳、可回溯；代价是 GLM 有调用成本（本地兜底可控）。
