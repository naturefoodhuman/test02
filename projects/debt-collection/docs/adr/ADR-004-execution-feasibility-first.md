# ADR-004：策略必含执行可行性 + 回款概率
- 状态：已接受
- 日期：2026-06-12
- 背景：老板要点4——核心是实际讨回率；要避免"判了也执行不到老赖"。
- 决策：策略报告强制包含"执行可行性预判"段（对方失信/可执行财产/规避历史），优先非诉/财产保全等高回款手段，不给脱离国内执行实际的纸面方案。
- 后果：贴近落地；代价是需要更多取数支撑（已有取数层）。
