# ADR-001：本地 CLI + SQLite 形态
- 状态：已接受
- 日期：2026-06-12
- 背景：债务数据高度敏感，个人单用户，V1 需最简、隐私最佳。
- 决策：做本地 CLI + SQLite，纯本地存储，不做 Web、不上云。
- 备选：Web 应用（弃因：增加复杂度+隐私风险）；JSON 文件（弃因：多笔查询弱）。
- 后果：隐私好、简单；代价是无 GUI（V1 可接受，后续可加）。
